# Reinforcement learning
# Kaboom
Play Kaboom (Atari 2600) with Reinforcement Learning

# Setup
1.ใช้ Python 64-bit

#Python Library
ติดตั้ง Gym
~~~{.python}
pip install gym
~~~
หรือ
~~~{.python}
git clone https://github.com/openai/gym
cd gym
pip install -e .
~~~

Install dependency
~~~{.python}
pip3 install -r requirements.txt
~~~

2. Import Roms
~~~{.python}
python3 -m retro.import /path/to/your/ROMs/directory/
~~~
